# Linux
- Linux 是一种自由和开放源码的类 UNIX 操作系统。
- Linux 英文解释为 Linux is not Unix。
- Linux 是在 1991 由林纳斯·托瓦兹在赫尔辛基大学上学时创立的，主要受到 Minix 和 Unix 思想的启发。
- Linux继承了Unix以网络为核心的设计思想，是一个性能稳定的多用户网络操作系统。
